# 🧠 CreateFlowchart.com

**CreateFlowchart.com** is an AI-powered flowchart platform designed to help builders, product managers, and developers create, improve, and automate workflows instantly.

Unlike static drawing tools, CreateFlowchart combines the speed of **Next.js 16.2.1** with the intelligence of **AI-driven logic analysis** and the reliability of **Yjs-powered real-time collaboration**.

---

## ⚡ Core Pillars

- **Prompt-to-Flow**: Generate complex logic diagrams and process flows from simple text descriptions.
- **Logic Improvement**: AI-assisted thinking that detects dead ends, loops, and logic errors in real-time.
- **Real-time Multiplayer**: Buttery-smooth collaborative editing with live cursors and shared awareness.
- **Action & Automation**: High-fidelity visualization connected to external webhooks and process automation nodes.
- **Semantic Discovery**: A community-driven gallery and template library powered by AI vector search (pgvector).

---

## 🚀 High-Level Plans

### Phase 1: The Intelligent Editor
- [ ] Universal **FlowGraph JSON** engine for deterministic rendering.
- [ ] "Sandbox Mode" for instant, auth-free flowcharting.
- [ ] AI Confidence Scoring to highlight logic weaknesses.

### Phase 2: Collaboration & Sharing
- [ ] Real-time cursor sync and multi-user editing (Yjs + Standalone Realtime Server).
- [ ] High-fidelity exports for SVG, PNG, PDF, and Mermaid formats.
- [ ] **Community Gallery**: Showcase and remix logic patterns from top builders.

### Phase 3: Flow Automation
- [ ] **Action Nodes**: Trigger external APIs directly from your flowchart branches.
- [ ] Serverless Flow Runner to execute logic as a service.
- [ ] Advanced 3D visualization for complex process mapping.

---

## 🛠 Tech Stack
- **Frontend**: Next.js 16.2.1 (App Router), Zustand, React 19.
- **Real-time**: Node.js, Yjs (CRDT), WebSockets.
- **Database**: Postgres 16 (pgvector) + Redis.
- **Orchestration**: Coolify (Dockerized Self-Hosted PaaS).
