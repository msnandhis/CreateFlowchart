# `@createflowchart/core`

Legacy FlowGraph schema, rules, and compatibility transforms.

## Install

```bash
pnpm add @createflowchart/core
```

## Use

```ts
import { FlowGraphSchema, validateFlowGraph } from "@createflowchart/core";
```

Use this package when you need the original FlowGraph model or compatibility helpers while migrating toward `DiagramDocument`.
