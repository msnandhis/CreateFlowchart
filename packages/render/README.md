# `@createflowchart/render`

Document-first SVG rendering for CreateFlowchart diagrams.

## Install

```bash
pnpm add @createflowchart/render @createflowchart/schema
```

## Use

```ts
import { renderDocumentToSvg, getDocumentBounds } from "@createflowchart/render";
```

Use this package for export, embeds, previews, and static rendering pipelines.
